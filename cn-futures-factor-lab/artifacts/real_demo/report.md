# 真实数据基础演示回测报告

> 本报告使用授权导出的真实历史数据，但仍只是工程链路和基础因子的技术演示，已加入基础可成交性、时间分期和成本压力检查，但尚未补齐完整交易规则与正式模型实验。

## 运行摘要

- 数据类型：`real`
- 输入来源：`/Users/Wantree/Desktop/AI金融实训营项目/cn-futures-factor-lab/data/interim/sufe_contract_daily.parquet`
- 输入记录数：764,863
- 商品品种数：49
- 信号因子：`momentum_20`
- 回测区间：2020-02-11 至 2025-12-30
- 有效交易日：1432
- 执行时间线：t 日收盘后形成信号 → t+1 交易日执行
- 收益定义：执行日目标权重乘以所选具体合约从执行日到下一交易日的结算价收益
- 成本：普通调仓成本 + 主力换月额外成本
- 指标口径：`annual_return` 为 CAGR；`sharpe` 为日收益算术均值/标准差 × √252

## 解释边界

1. 零 OHLC/零成交记录只保留日期链，公共执行层会拒绝在这些记录上形成目标仓位；
2. 第一版使用日频结算价，没有模拟涨跌停、保证金追缴和日内成交冲击；
3. 当前用三档成本压力代替未知的精确历史费率；正式研究仍应补充品种级手续费、合约乘数和历史交易规则；
4. 机器学习只能用训练期拟合、验证期选参；测试期和新增最终盲测不得参与调参。
