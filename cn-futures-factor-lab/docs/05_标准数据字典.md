# 标准数据字典

## 1. 合约日行情

主键：`trade_date + contract`

| 字段 | 类型 | 必需 | 含义 |
|---|---|---:|---|
| `trade_date` | datetime | 是 | 归属交易日，不一定等于夜盘自然日期 |
| `exchange` | string | 是 | SHFE、DCE、CZCE、INE、GFEX 等 |
| `product` | string | 是 | 品种代码，如 RB、CU、M |
| `contract` | string | 是 | 可交易月份合约代码 |
| `open` | float | 是 | 开盘价 |
| `high` | float | 是 | 最高价 |
| `low` | float | 是 | 最低价 |
| `close` | float | 是 | 收盘价 |
| `settlement` | float | 是 | 当日结算价 |
| `pre_settlement` | float | 否 | 前一交易日结算价 |
| `volume` | float | 是 | 成交量，统一为手或张 |
| `open_interest` | float | 是 | 持仓量，统一为手或张 |
| `turnover` | float | 否 | 成交额，统一单位后使用 |
| `upper_limit` | float | 否 | 涨停价 |
| `lower_limit` | float | 否 | 跌停价 |
| `source` | string | 否 | 数据来源标识 |

## 2. 主力连续面板

| 字段 | 含义 |
|---|---|
| `selection_value` | 用于选择主力的滞后持仓量或成交量 |
| `delivery_yyyymm` | 交割年月 |
| `previous_contract` | 上一交易日选中的主力合约 |
| `is_roll_day` | 当日是否发生主力切换 |
| `contract_return` | 当前具体合约自身的当日收益 |
| `forward_return` | 当前具体合约从当日至下一日的收益 |
| `continuous_return` | 用于构造连续研究价格的收益 |
| `continuous_close` | 以 100 起始的连续研究指数 |

## 3. 因子与标签

| 字段 | 含义 |
|---|---|
| `momentum_20` | 过去 20 日连续价格收益 |
| `momentum_60` | 过去 60 日连续价格收益 |
| `reversal_5` | 过去 5 日收益的相反数 |
| `volatility_20` | 过去 20 日日收益年化波动率 |
| `liquidity_20` | 过去 20 日平均成交量的对数 |
| `open_interest_change_20` | 过去 20 日持仓量变化率 |
| `target_return_1d` | 未来 1 日连续价格收益标签 |
| `target_return_5d` | 未来 5 日连续价格收益标签 |
| `target_return_20d` | 未来 20 日连续价格收益标签 |

## 4. 后续需要补充的静态数据

真实回测应增加：合约上市日、最后交易日、交割日、合约乘数、最小变动价位、手续费规则、保证金比例和交易时段。静态参数也可能随历史变化，不能只保存当前值。
