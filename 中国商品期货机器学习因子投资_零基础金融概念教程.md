# 中国商品期货机器学习因子投资：零基础金融概念教程

> 面向完全不了解商品期货、因子投资和量化金融的读者。  
> 本文只解释金融概念、研究逻辑和项目要回答的问题，暂不讨论代码、数据库、程序架构或部署。

## 0. 先用一句话理解这个项目

这个项目想做的事情是：

> 每隔一段时间，观察一批中国商品期货的价格趋势、近远月价格关系、成交与持仓活跃度、波动风险等特征，让传统方法和机器学习分别判断哪些品种未来相对更强、哪些相对更弱，然后检验“买入预测较强的品种、卖出预测较弱的品种”能否在未参与模型训练的未来时期获得稳定的成本后收益。

这句话包含五个关键词：

1. **一批商品期货**：研究对象不是一只股票，而是铜、铝、螺纹钢、原油、豆粕、白糖等多个品种。
2. **特征或因子**：用数字描述每个品种当时的状态，例如过去三个月涨幅、期限结构、波动率和流动性。
3. **横截面比较**：在同一天比较不同品种，而不仅是预测某一个品种明天涨不涨。
4. **机器学习组合**：让模型学习多个因子与未来收益之间可能存在的非线性关系。
5. **样本外和成本后检验**：只能用当时已知的信息做判断，还要扣除手续费、滑点和换月成本。

如果把项目类比成考试：历史数据是练习题，验证期是模拟考试，测试期是真正的闭卷考试。只有闭卷考试表现好，模型才可能具有研究价值。

### 0.1 推荐学习顺序

不要试图一次记住所有公式。建议按下面顺序阅读：

1. 第1—4章：先知道期货怎样交易，尤其理解保证金、合约月份、主力合约和换月；
2. 第5—6章：理解商品期货独有的基差和期限结构；
3. 第7—12章：理解什么是横截面因子投资，以及怎样评价一个因子；
4. 第13章：理解机器学习在研究中承担的角色；
5. 第14—16章：理解组合、风险、回测和研究偏差；
6. 第17—22章：把全部概念串成一个完整项目。

## 1. 开始前必须知道的最基础金融概念

### 1.1 资产、价格与收益率

**资产**是能够被持有、交易并可能带来未来经济利益的东西。股票、债券、基金、商品期货合约都属于金融资产或金融工具。

**价格**是某个时点买卖资产所需的金额。研究投资表现时，更常用的是**收益率**，因为不同品种的价格单位不同，直接比较价格涨跌金额没有意义。

最简单的收益率是：

\[
R_{t+1}=\frac{P_{t+1}-P_t}{P_t}
\]

其中：

- \(P_t\)：今天的价格；
- \(P_{t+1}\)：下一期的价格；
- \(R_{t+1}\)：从今天到下一期的收益率。

例如，价格从100元涨到103元，收益率为3%。

量化研究也常使用**对数收益率**：

\[
r_{t+1}=\ln(P_{t+1})-\ln(P_t)
\]

在日收益很小时，普通收益率和对数收益率非常接近。对数收益率便于做时间序列计算，但实际组合财富累计通常仍需使用普通收益率。

### 1.2 风险不是“肯定亏钱”

金融中的风险通常指未来结果的不确定性，而不只是亏损。

常见风险包括：

- **价格风险**：市场价格向不利方向变化；
- **波动风险**：价格变化幅度突然变大；
- **流动性风险**：想交易时没有足够对手盘，或只能接受很差的价格；
- **杠杆风险**：较小的价格变化导致较大的资金损益；
- **模型风险**：模型在历史上有效，在未来失效；
- **操作风险**：合约选择、换月、数据或程序处理错误。

### 1.3 做多与做空

**做多（Long）**：先买入，期望价格上涨后卖出获利。

**做空（Short）**：先建立卖出方向的仓位，期望价格下跌后平仓获利。期货天然允许做空，不需要像股票融券那样先借到证券。

假设一份合约价格从100跌到95：

- 多头亏损5个价格单位；
- 空头盈利5个价格单位。

多头与空头的损益方向相反，但实际交易还需考虑手续费、滑点和保证金。

### 1.4 名义本金、保证金与杠杆

一份期货合约代表的经济价值称为**合约名义价值**：

\[
\text{名义价值}=\text{期货价格}\times\text{合约乘数}
\]

例如，某期货价格为每吨4,000元，一手代表10吨，则一手名义价值为40,000元。

交易者通常不需要支付全部40,000元，只需缴纳一部分作为**保证金**。如果保证金比例为10%，初始占用资金约为4,000元。由此形成了**杠杆**：用较少资金承担较大名义价值的价格风险。

杠杆不会自动提高策略质量，只会同时放大盈利和亏损。研究报告必须区分：

- 合约名义敞口；
- 实际保证金占用；
- 账户资金；
- 策略使用的目标杠杆。

### 1.5 逐日盯市与平仓

期货通常实行**逐日盯市**：交易所每天按结算价计算持仓盈亏，并将盈亏计入保证金账户。

**开仓**是建立新头寸，**平仓**是用相反方向交易结束原头寸。若账户权益低于要求，可能需要补充保证金；风险过大时可能被强制平仓。

## 2. 什么是商品、现货和商品期货

### 2.1 什么是商品

这里的“商品”不是商场里的普通商品，而是可以标准化交易的大宗商品，例如：

- 能源：原油、燃料油；
- 金属：铜、铝、黄金、白银；
- 黑色产业链：铁矿石、焦煤、焦炭、螺纹钢；
- 化工：PTA、聚乙烯、聚丙烯、甲醇；
- 农产品：大豆、豆粕、豆油、玉米、白糖、棉花。

商品价格受供给、需求、库存、运输、天气、政策、汇率和全球贸易等因素影响。

### 2.2 现货

**现货（Spot）**是当前或近期完成交付的真实商品。现货价格反映某一地区、品质、交货地点和时间条件下的即时供需。

“铜的现货价格”并不是天然唯一的数字，因为不同地区、品牌、品质和交付条件可能产生不同报价。

### 2.3 期货

**期货合约（Futures Contract）**是交易所制定的标准化合约，约定在未来某一月份，以规则确定的方式买卖一定数量和质量的标的物。

标准化内容通常包括：

- 标的商品和质量标准；
- 一手对应的数量，即合约乘数；
- 报价单位；
- 最小价格变动单位；
- 到期或交割月份；
- 交易时间；
- 涨跌停板；
- 最低保证金；
- 交割地点和交割方式。

绝大多数量化策略会在进入交割环节前平仓或换月，并不真正接收几吨铜或几百桶原油。

### 2.4 商品期货存在的基本作用

商品期货主要有三类经济功能：

1. **风险管理或套期保值**：生产商、加工商和消费企业提前锁定价格。
2. **价格发现**：大量市场参与者把对未来供需的判断反映到期货价格中。
3. **投资与投机**：交易者根据价格判断承担风险，希望获得收益。

### 2.5 套期保值与投机

**套期保值（Hedging）**的目标是降低实体经营风险。例如，航空公司担心未来燃油价格上涨，可以建立与燃油价格正相关的期货多头。

**投机（Speculation）**是主动承担价格风险，希望从方向判断中获利。

本项目构建的横截面多空组合更接近系统化投资或投机，但通过多品种分散、风险控制和多空配对降低部分市场方向风险。

### 2.6 套利

**套利（Arbitrage）**严格意义上指利用相同或高度相关资产之间的不合理价差，构造低风险甚至理论无风险的交易。

现实中的跨期、跨品种和期现策略通常仍然有基差、流动性、交割和模型风险，因此很多被称为“套利”的策略并非真正无风险。

## 3. 一份期货合约怎样被描述

### 3.1 品种与合约不是一回事

“螺纹钢”是一个**期货品种**，而“某年10月到期的螺纹钢期货”才是一份具体**合约**。

同一品种在同一天通常存在多个到期月份的合约。例如：

- 螺纹钢近月合约；
- 螺纹钢次近月合约；
- 更远月份的螺纹钢合约。

因此，一张数据表中可能同时有很多行属于同一商品，但交割月份不同。

### 3.2 开盘、最高、最低、收盘与结算价

- **开盘价（Open）**：某交易日开始阶段形成的价格；
- **最高价（High）**：当日最高成交价；
- **最低价（Low）**：当日最低成交价；
- **收盘价（Close）**：交易日结束阶段的成交价格；
- **结算价（Settlement）**：交易所按规则计算、用于每日盈亏结算的价格。

收盘价与结算价可能不同。研究必须明确因子计算、收益计算和实际成交分别使用什么价格。

### 3.3 成交量、成交额与持仓量

**成交量（Volume）**：某段时间内成交的合约数量。

**成交额（Amount）**：成交合约对应的金额规模，具体统计口径需以数据源为准。

**持仓量（Open Interest）**：交易结束后仍未平仓的合约数量。

直观区别：

- 成交量描述“今天交易得多不多”；
- 持仓量描述“市场上仍有多少未结束的合约头寸”。

价格、成交量和持仓量联合变化常被用来观察资金进入、趋势确认和市场拥挤，但不存在任何永远正确的简单组合规则。

### 3.4 合约乘数与最小变动价位

**合约乘数**决定一手合约代表多少商品。

**最小变动价位（Tick Size）**决定价格一次最少可以变化多少。

每跳价值为：

\[
\text{每跳价值}=\text{最小变动价位}\times\text{合约乘数}
\]

它直接影响实际盈亏和交易成本估计。

### 3.5 涨跌停板

交易所通常限制合约一天内相对参考价格的最大涨跌幅，这就是**涨跌停板**。

涨停并不代表一定能按涨停价卖出，跌停也不代表一定能按跌停价买入。若市场只有单边挂单，策略可能无法成交。因此回测不能看到信号后就假设总能按理想价格完成交易。

## 4. 主力合约、近远月、连续合约和换月

这是商品期货研究中最容易被初学者忽视、却最重要的一组概念。

### 4.1 近月与远月合约

- **近月合约**：距离到期较近；
- **远月合约**：距离到期较远；
- **次近月合约**：到期顺序排在近月之后。

近月不一定就是成交最活跃的合约。某些品种的交易可能集中在特定月份。

### 4.2 主力合约

**主力合约**通常指某品种中成交量或持仓量最大的合约。它不是永远固定的，会随时间从旧合约切换到新合约。

主力合约定义必须事先固定，例如：

- 当日成交量最大；
- 当日持仓量最大；
- 新合约连续若干日超过旧合约后才切换。

不同规则可能产生不同收益，因此必须做稳健性检验。

### 4.3 为什么需要连续合约

单个期货合约会到期，不能直接提供十年的连续历史。研究者常把历任主力或近月合约拼接成一条**连续合约序列**。

连续合约是研究用途的人工序列，并不是交易所中真正存在的一份可永久交易合约。

### 4.4 换月或移仓

当旧合约临近交割或失去流动性时，投资者需要：

1. 平掉旧合约；
2. 建立新合约仓位。

这称为**换月、移仓或展期（Roll）**。

换月会产生：

- 旧合约平仓成本；
- 新合约开仓成本；
- 两合约价格差带来的展期影响；
- 可能无法同时成交的执行风险。

### 4.5 连续序列与可交易收益必须分开

为了让价格图平滑，数据商可能对历史价格做前复权、后复权或价差调整。这些调整有利于计算趋势，却不等于真实财富变化。

正确理解是：

- 连续序列用于构造历史特征；
- 真实的逐合约价格用于计算交易和换月收益。

如果直接用复权连续价格计算策略收益，可能制造并不存在的利润。

## 5. 基差、期限结构、Contango与Backwardation

### 5.1 基差

一种常见定义是：

\[
\text{基差}=\text{现货价格}-\text{期货价格}
\]

但有些数据源使用“期货减现货”。看到基差字段时，第一件事必须是确认符号定义。

为便于不同价格尺度的品种比较，可使用**基差率**：

\[
\text{基差率}=\frac{S_t-F_t}{S_t}
\]

其中 \(S_t\) 为现货价格，\(F_t\) 为期货价格。

基差会受到仓储、资金成本、便利收益、交割制度、地区与品质差异影响，因此不能简单理解为“期货价格错了”。

### 5.2 期货曲线或期限结构

把同一商品不同到期月份的期货价格按到期时间排列，就得到**期货曲线或期限结构（Term Structure）**。

例如：

| 到期时间 | 近月 | 次近月 | 第三合约 | 第四合约 |
|---|---:|---:|---:|---:|
| 价格 | 100 | 102 | 103 | 104 |

这是一条总体向上的曲线。

### 5.3 Contango与Backwardation

**Contango（升水型期限结构）**：远月价格通常高于近月价格，曲线向上。

**Backwardation（贴水型期限结构）**：近月价格通常高于远月价格，曲线向下。

常见经济解释包括：

- 库存高、仓储和融资成本突出时，曲线更可能向上；
- 现货紧张、立即持有商品价值较高时，近月价格可能更贵，曲线更可能向下。

这些只是经济直觉，不是机械定律。

### 5.4 持有成本与便利收益

理论上，期货和现货之间受到以下因素影响：

- 无风险利率或资金成本；
- 仓储费；
- 保险和运输等持有成本；
- **便利收益（Convenience Yield）**。

便利收益是实际持有现货带来的非现金好处。例如，工厂持有原材料可以避免生产中断。当现货非常稀缺时，这种好处可能很高。

### 5.5 Carry与展期收益

**Carry**可以理解为持有某个期货头寸随时间推进所面对的期限结构收益线索。研究中常用近远月价差并按到期时间年化：

\[
Carry_{i,t}
=
\frac{\ln F_{near,i,t}-\ln F_{far,i,t}}
{\tau_{far}-\tau_{near}}
\]

其中 \(\tau\) 表示距离到期的时间。不同论文的符号方向可能相反，必须先固定项目定义。

**展期收益（Roll Yield）**是旧合约向新合约换月时，期限结构变化带来的收益或损失。它与Carry有关，但两者不是在任何定义下都完全相同。

直观上：

- 若近月高于远月，做多者换入较便宜的远月，可能获得有利展期；
- 若远月高于近月，做多者换入较贵的远月，可能承受不利展期。

实际结果还取决于曲线之后怎样变化。

### 5.6 价差交易

**跨期价差**是在同一品种的不同到期合约上建立相反方向头寸。例如：买近月、卖远月。

它主要暴露于期限结构变化，而不是商品整体价格水平变化。价差的风险通常低于单边头寸，但绝不是无风险。

## 6. 期限结构的水平、斜率与曲率

参考资料使用Nelson-Siegel思想，用三个参数概括整条期货曲线。

### 6.1 水平 Level

水平描述整条曲线大致位于什么价格区域。若不同期限价格一起上涨，可视为曲线整体上移。

### 6.2 斜率 Slope

斜率描述近月相对远月是更贵还是更便宜：

- 更明显的Backwardation是一种斜率状态；
- 更明显的Contango是另一种斜率状态。

斜率变化反映供需紧张程度、库存状态或预期的变化。Bianchi等的研究发现，近期斜率变化的延续在其全球商品样本中具有较强的策略表现，并且比曲率策略更能承受合理交易成本。

### 6.3 曲率 Curvature

曲率描述曲线中间部分是否相对凸起或凹下。它需要至少三个、通常四个不同期限的合约才能有意义。

围绕曲率可构造**蝶式价差**，例如对近月、次近月和更远月建立不同方向和数量的组合。蝶式策略交易腿数较多，对流动性和成本更敏感。

### 6.4 Nelson-Siegel模型的直观意义

Nelson-Siegel模型最初常用于收益率曲线。用于商品时，它尝试用少数参数拟合同一商品多个到期合约的价格曲线。

对本项目而言，重要的不是记住完整公式，而是理解：

> 原本需要观察四个甚至更多合约价格，现在可以用“水平、斜率、曲率”三个数字概括，并把这些数字或其变化作为因子。

## 7. 什么是因子投资

### 7.1 因子

**因子（Factor）**是根据可观察数据计算出的数字，用于描述资产某一方面的特征，并尝试解释或预测未来收益。

例如，三个月动量因子可以写成：

\[
Momentum_{i,t}=\frac{P_{i,t}}{P_{i,t-60}}-1
\]

其中 \(i\) 表示品种，约60个交易日代表三个月。

因子不是“保证赚钱的公式”。一个合格因子至少应具备：

- 清晰的计算定义；
- 可解释的经济逻辑；
- 只使用当时可得的数据；
- 稳定的样本外预测能力；
- 扣除交易成本后的经济价值。

### 7.2 信号、特征与因子的区别

在实践中三者经常混用：

- **特征（Feature）**：模型输入的任何变量；
- **因子**：有一定金融含义、用于解释或预测收益的特征；
- **交易信号（Signal）**：直接决定买卖方向和仓位的最终数值。

例如：过去三个月收益是因子；模型把20个因子组合成预测分数，这个分数是交易信号。

### 7.3 风险溢价与错误定价

因子为什么可能赚钱，通常有两类解释：

1. **风险补偿**：承担别人不愿承担的风险，所以长期获得更高收益。
2. **错误定价或行为偏差**：投资者反应不足、反应过度或受到情绪影响，价格暂时偏离合理水平。

同一个因子可能同时包含两类机制，很难完全区分。

### 7.4 Alpha与Beta

**Beta**通常指对系统性风险的暴露。例如，组合上涨可能只是因为整个商品市场上涨。

**Alpha**指在控制已知风险暴露后仍无法被解释的超额收益。

项目不能只看组合赚没赚钱，还要问：

- 是否只是长期做多商品？
- 是否集中在某个板块？
- 是否只是承担了更高波动和流动性风险？
- 控制动量、Carry等传统因子后是否仍有增量收益？

## 8. 时间序列与横截面

### 8.1 时间序列问题

时间序列研究同一个品种随时间的变化。例如：

> 根据铜过去一年的价格，预测铜下周上涨还是下跌。

比较对象是“铜的今天与铜的过去”。

### 8.2 横截面问题

横截面研究同一时点不同品种之间的差异。例如：

> 在今天的30个商品品种中，哪些下周相对更强，哪些相对更弱？

比较对象是“今天的铜、铝、原油、豆粕等品种”。

本项目的核心是**横截面收益预测**。

### 8.3 面板数据

同时包含品种维度和时间维度的数据称为**面板数据（Panel Data）**。

| 日期 | 品种 | 动量 | Carry | 波动率 | 未来5日收益 |
|---|---|---:|---:|---:|---:|
| T日 | 铜 | 0.08 | 0.03 | 0.21 | 0.012 |
| T日 | 豆粕 | -0.02 | 0.05 | 0.18 | -0.006 |
| T日 | 原油 | 0.11 | -0.04 | 0.35 | 0.020 |

模型利用大量“日期×品种”的观测，学习当时因子与之后收益之间的关系。

### 8.4 横截面排序

假设同一天有五个品种，其动量分别为：

| 品种 | 动量 |
|---|---:|
| 铜 | 8% |
| 铝 | 3% |
| 原油 | 12% |
| 豆粕 | -2% |
| 白糖 | 1% |

按动量从高到低排列：原油、铜、铝、白糖、豆粕。动量策略可能买入排名靠前的品种并卖出排名靠后的品种。

横截面排序比直接比较原始数值更稳健，因为不同商品的价格尺度和波动特征不同。

### 8.5 相关系数与协方差

**相关系数（Correlation）**衡量两个收益序列共同变化的方向和强度，通常位于-1到1之间：

- 接近1：经常同向变化；
- 接近-1：经常反向变化；
- 接近0：线性共同变化较弱。

**协方差（Covariance）**也衡量共同变化，但保留了变量尺度。投资组合风险计算常使用协方差矩阵。

相关性会随时间变化，也不代表因果关系。图神经网络可以把滚动相关性作为品种之间的边，但模型学到的边仍只是统计关系。

## 9. 本项目中的主要因子

### 9.1 动量因子

**动量（Momentum）**表示过去表现较强的资产在未来一段时间仍可能相对较强。

常见窗口：

- 1个月；
- 3个月；
- 6个月；
- 12个月；
- 12个月收益但跳过最近1个月。

可能解释包括趋势跟随、信息反应不足和资金持续流入。动量也可能突然反转，因此常伴随“动量崩溃”风险。

### 9.2 短期反转因子

**短期反转（Short-term Reversal）**认为短期急涨或急跌中有一部分来自暂时性订单冲击，之后可能修复。

它与中期动量并不矛盾：一个品种可以在几个月尺度上保持上涨趋势，同时在未来几天出现回调。

### 9.3 基差、Carry与展期因子

这类因子利用现货与期货、近月与远月之间的价格关系，反映库存、资金成本和供需紧张程度。

它们是商品期货区别于普通股票因子的重要特征，应作为本项目核心因子。

### 9.4 流动性因子

**流动性（Liquidity）**表示资产能否以较小价格冲击快速成交。

常见代理变量：

- 成交量；
- 成交额；
- 持仓量；
- 成交量或持仓量的变化；
- 换手率；
- Amihud非流动性指标。

Amihud指标的直观形式是：

\[
ILLIQ=\frac{|R_t|}{Amount_t}
\]

相同成交额引起的价格变化越大，通常表示流动性越差。

流动性差的品种可能提供风险补偿，但也更难把理论收益真正交易出来。

### 9.5 波动率因子

**波动率（Volatility）**衡量收益率变化的幅度。常用历史收益的标准差估计：

\[
\sigma=\sqrt{\frac{1}{n-1}\sum_{k=1}^{n}(r_k-\bar r)^2}
\]

波动率高不代表预期收益一定高，也不代表一定会跌；它只说明结果更不稳定。

### 9.6 下行波动率

**下行波动率**只关注负收益或低于目标收益的部分，比普通波动率更接近投资者对亏损的感受。

### 9.7 偏度与峰度

**偏度（Skewness）**描述收益分布是否向某一侧拖出长尾：

- 正偏：偶尔出现特别大的正收益；
- 负偏：偶尔出现特别大的负收益。

**峰度（Kurtosis）**描述极端变化出现的程度。商品收益常具有厚尾，意味着极端涨跌比正态分布假设下更常见。

### 9.8 最大回撤特征

**回撤**是从历史高点到之后低点的下降幅度。把某品种近期最大回撤作为因子，可以描述其近期下行压力或尾部风险。

### 9.9 量价因子

量价因子联合使用价格、成交量、成交额和持仓量，例如：

- 价格上涨是否伴随成交量放大；
- 价格趋势是否得到持仓量增加确认；
- 成交量是否突然偏离历史均值；
- 高低价振幅是否伴随流动性变化。

量价关系可能反映资金进入、交易拥挤或信息冲击，但需要避免把经验口诀当成必然规律。

### 9.10 库存、仓单与现货因子

**库存**反映可用商品存量。

**仓单**是符合交易所交割标准、经注册的货物凭证。仓单变化可能反映可交割供应变化，但仓单不等于社会全部库存。

现货价格、库存、仓单和基差联合起来，通常比只看期货价格更接近真实供需。

### 9.11 天气因子

天气会影响农产品产量、能源需求和运输。天气因子可能包括温度、降水、干旱指数和异常天气。

使用天气数据时必须确定：

- 哪个产区与哪个品种相关；
- 数据在当时何时发布；
- 使用的是实时观测还是后来修订的数据；
- 季节性是否被正确处理。

### 9.12 产业链因子

商品之间存在生产、加工、替代和共同成本关系。例如：

```text
铁矿石 + 焦煤/焦炭 → 钢材
原油 → 化工品
大豆 → 豆粕 + 豆油
```

**产业链因子**尝试利用上下游价差、替代品价格或同板块共同变化预测收益。

相关性不等于因果关系。两种商品一起上涨，可能只是共同受到美元、宏观需求或市场情绪影响。

### 9.13 市场状态与状态依赖

因子可能只在某些**市场状态（Market Regime）**下有效，例如：

- 牛市、熊市或震荡市；
- 高波动或低波动；
- 高库存或低库存；
- 流动性充裕或紧张；
- 政策冲击前后。

**状态依赖**指同一因子的方向或强度随环境变化。机器学习可能捕捉这种差异，但状态分类本身也必须只使用当时可得的信息。

## 10. 因子在进入模型前为什么要处理

### 10.1 异常值与缩尾

极端数据可能来自真实行情，也可能来自错误价格、合约切换或数据缺失。

**缩尾（Winsorization）**不是删除整条记录，而是把过于极端的数值压到某个边界，降低少数异常值对模型的支配。

### 10.2 标准化

不同因子单位不同。例如动量是百分比，成交额可能是亿元。**标准化**把它们转换到可比较尺度。

常见Z-score：

\[
z_{i,t}=\frac{x_{i,t}-\bar x_t}{s_t}
\]

这里的均值和标准差在同一天的品种横截面上计算。

### 10.3 排名化

**排名化**把原始因子转换为品种在当日横截面的相对名次。它对极端值更不敏感，也更符合横截面选品种的目标。

### 10.4 中性化

如果模型买入的全部是黑色品种，它赚到的可能只是黑色板块行情，而不是因子本身。

**板块中性化**尝试去除能源、金属、化工、农产品等板块共同影响。还可以做波动率、流动性等中性化。

中性化的目的不是让所有风险消失，而是让研究者更清楚收益来自哪里。

### 10.5 因子方向

某些因子越大越看多，某些越大越看空。因子的正负方向必须只用历史数据决定，不能观察完整测试期后再把表现差的因子整体反向。

## 11. 如何判断一个因子有没有预测能力

### 11.1 信息系数 IC

**信息系数（Information Coefficient, IC）**衡量某日因子值与之后收益之间的横截面相关性。

本项目通常使用Spearman秩相关，也称**Rank IC**：它关注排名是否一致，而不是原始数值是否线性对应。

直观理解：

- IC为正：因子排名高的品种之后倾向表现更好；
- IC为负：因子排名高的品种之后倾向表现更差；
- IC接近0：当期几乎没有横截面排序能力。

单日IC很不稳定，因此要看长期均值和分布。

### 11.2 ICIR

**ICIR**衡量IC的平均水平相对于波动的大小：

\[
ICIR=\frac{Mean(IC)}{Std(IC)}
\]

IC均值相同的两个因子中，IC波动较小的通常更稳定。

### 11.3 分组检验

把品种按因子从低到高分成五组或十组，观察未来平均收益是否逐级上升或下降。

这称为**分组收益单调性**。如果只有最极端的一组偶尔大赚，中间组完全无规律，因子可能不够稳定。

### 11.4 多空组合

买入因子排名最高的一组、卖出排名最低的一组，形成**多空组合（Long-Short Portfolio）**。

其收益近似为：

\[
R^{LS}_{t+1}=R^{Long}_{t+1}-R^{ShortGroup}_{t+1}
\]

其中 \(R^{ShortGroup}\) 是被卖空组资产本身的收益。若空头组下跌，减去负数就形成正贡献。

多空组合希望降低整个商品市场共同涨跌的影响，突出相对选品种能力，但并不天然做到完全市场中性或风险中性。

### 11.5 统计显著性与经济显著性

**统计显著性**问：观察到的结果是否可能只是随机波动。

**经济显著性**问：结果扣除实际成本后是否足够大，是否值得承担风险和占用资金。

一个因子可以统计显著但赚得太少，也可以收益看似很高但样本太短、统计上不可靠。

### 11.6 Newey-West调整

金融收益常存在自相关和波动变化，普通t统计量可能过于乐观。**Newey-West标准误**用于对这类问题进行一定修正。

它不是万能工具，但比直接使用独立同分布假设更符合金融时间序列特点。

### 11.7 MAE、RMSE与样本外R²

这些指标评价预测值与真实值之间的误差：

- **MAE**：绝对预测误差的平均值，直观且对极端值相对不敏感；
- **RMSE**：平方误差平均后开根号，对大误差处罚更重；
- **样本外 \(R^2\)**：模型相对一个简单基准是否降低了样本外预测误差。

样本外 \(R^2\) 为负并不表示计算错误，而是说明模型还不如基准预测。金融收益难预测，不能只选择最有利的指标汇报。

### 11.8 方向准确率

**方向准确率**是上涨和下跌方向判断正确的比例。它没有考虑每次涨跌的幅度：模型可能多次猜对很小的涨跌，却在一次巨大行情中判断错误并造成严重亏损。

### 11.9 Diebold-Mariano检验

**Diebold-Mariano检验**用于比较两个预测模型的损失是否存在显著差异。例如，检验LightGBM的样本外绝对误差是否显著低于线性模型。

它回答的是预测误差差异，而不是直接证明某个交易组合可以赚钱。

### 11.10 Bootstrap置信区间

**Bootstrap**通过对历史观测进行重复抽样，估计收益、IC或夏普比率的不确定范围。

金融时间序列存在时间依赖时，通常需要按时间块抽样，而不能随意逐日打乱。

### 11.11 False Discovery Rate

同时检验很多因子时，部分“显著结果”会偶然出现。**False Discovery Rate（FDR，错误发现率）**控制方法试图限制被宣布有效的因子中，预期有多少比例是假发现。

它适合因子动物园和LLM批量生成因子场景。

## 12. 从单因子到多因子

### 12.1 为什么组合因子

任何单个因子都可能在某些时期失效。动量、Carry、流动性和风险因子可能包含不同信息。组合因子的目的包括：

- 分散单因子失效风险；
- 利用不同因子的互补信息；
- 根据市场状态调整权重；
- 捕捉因子之间的交互。

### 12.2 等权多因子

最简单的方法是把所有标准化因子等权平均：

\[
Score_{i,t}=\frac{1}{K}\sum_{k=1}^{K}z_{i,t}^{(k)}
\]

它透明、稳定、难以过度调参，是机器学习必须击败的基准。

### 12.3 历史IC加权

根据因子过去一段时间的IC或ICIR分配权重。近期更有效的因子获得更高权重。

需要特别小心：权重只能使用当时之前的历史，不能用未来表现确定。

### 12.4 线性回归

线性模型假设未来收益是多个因子的加权和：

\[
R_{i,t+1}=\beta_0+\beta_1X_{1,i,t}+\cdots+\beta_KX_{K,i,t}+\varepsilon_{i,t+1}
\]

优点是易解释；缺点是难以描述阈值、弯曲关系和复杂交互。

### 12.5 Lasso与Elastic Net

因子较多且彼此相关时，普通回归容易不稳定。

- **Lasso**通过惩罚把部分系数压到0，相当于自动选择变量；
- **Elastic Net**结合Lasso和岭回归，对高度相关因子通常更稳定。

它们仍然是线性模型，但比普通回归更适合多因子环境。

### 12.6 IPCA基准

Cheng等使用**工具变量主成分分析（Instrumented Principal Component Analysis, IPCA）**作为基准。它让资产对潜在公共因子的暴露随可观察特征变化。

初学阶段不需要推导IPCA。只需理解它比等权因子更复杂，可用于检验LLM因子组合是否包含传统特征型因子模型之外的信息。它不是本项目最小可行版本的必选模型。

## 13. 机器学习在本项目中做什么

### 13.1 监督学习

本项目主要属于**监督学习**：模型看到历史特征和之后真实发生的收益，学习二者之间的映射。

- 输入 \(X\)：动量、Carry、流动性、波动率等；
- 标签 \(y\)：之后5日或20日的收益或收益排名；
- 输出：每个品种的预测收益或预测分数。

### 13.2 非线性

线性模型认为因子每增加一个单位，影响大致固定。现实中可能存在：

- 动量只有超过一定强度才有效；
- 流动性过低时，理论收益无法交易；
- 波动率在中等水平有利，过高反而危险。

这类弯曲或分段关系称为**非线性关系**。

### 13.3 变量交互

一个因子的作用可能取决于另一个因子。例如：

> 动量较强且期限结构为Backwardation时，趋势可能更可信；但动量较强且流动性极差时，信号可能难以交易。

这称为**变量交互**。机器学习可能比简单加权更容易发现交互，但也更容易把噪声误当规律。

### 13.4 LightGBM与XGBoost

二者都属于**梯度提升树**。可以把它们理解为：连续训练许多较简单的决策树，每棵树重点修正前面模型的错误，最后汇总预测。

优势：

- 能处理非线性和交互；
- 对表格型因子数据通常有效；
- 不要求所有变量服从正态分布；
- 可以计算特征重要性和SHAP解释。

风险：

- 树太多、太深或反复调参会过拟合；
- 特征有泄漏时会非常“聪明”地利用泄漏；
- 高特征重要性不等于经济因果关系。

### 13.5 神经网络

神经网络通过多层变换学习复杂关系。它可能捕捉更复杂的模式，但通常需要更多数据，对超参数、随机种子和正则化更敏感。

中国商品期货的品种数量有限，日频横截面样本并不像互联网数据那样巨大。因此神经网络不一定优于树模型。

### 13.6 图神经网络 GNN

图神经网络把每个期货品种视为一个**节点**，把品种关系视为**边**。

边可以来自：

- 上下游产业链；
- 替代品关系；
- 同板块；
- 滚动相关性；
- 模型自动学习的动态关系。

图模型希望让某个品种的预测同时参考相关品种的信息。

参考资料中的HCGNN还使用多任务学习，同时预测价格、移动平均、价格区间和变化点。但其高频样本与测试期很短，所以适合借鉴结构，不应直接照搬其收益结论。

### 13.7 多任务学习

**多任务学习**让一个模型同时学习多个相关目标，希望不同任务相互提供信息。

例如同时学习：

- 未来价格；
- 移动平均趋势；
- 高低价区间；
- 趋势变化点。

如果任务确实相关，可能提高泛化能力；如果任务定义牵强，也可能互相干扰。

### 13.8 持续学习与灾难性遗忘

**持续学习（Continual Learning）**指模型按顺序学习新任务或新时期。

神经网络学习新任务后可能破坏旧任务中有用的参数，这称为**灾难性遗忘**。参考资料中的HCGNN使用参数重要性和互信息思路缓解这一问题。

对本项目主线而言，这属于进阶概念，不是完成基础因子投资研究的必要条件。

### 13.9 LLM生成因子

大语言模型可以根据字段说明提出因子公式和代码。例如组合动量、波动率、成交量和基差。

它的合理定位是：

> 帮助人类提出候选假设，而不是替代金融逻辑、数据审计和样本外检验。

LLM可能产生：

- 使用未来数据的公式；
- 重复已有因子但换了名字；
- 除零、无穷值和异常稀疏结果；
- 股票市场逻辑生搬硬套到期货；
- 因提示词和温度变化而不稳定的结果。

因此必须记录提示词、保留全部候选因子、人工审核公式，并避免只汇报表现最好的少数结果。

### 13.10 SHAP与模型解释

**SHAP**是一种解释模型预测的方法，可以估计某个特征对某次预测的边际贡献。

它可以帮助回答：

- 模型主要依赖哪些因子；
- 动量在什么范围内贡献为正；
- Carry与流动性是否存在交互；
- 不同年份的重要因子是否变化。

SHAP解释的是模型如何使用数据，不证明经济因果。

## 14. 投资组合与风险控制概念

### 14.1 Long-only与Long-short

**Long-only（只做多）**组合只买入看好的品种，不建立空头。它容易受到整个商品市场涨跌影响。

**Long-short（多空）**组合同时做多高分品种、做空低分品种，更关注品种之间的相对表现。

“前10%”或“后10%”称为**十分位组（Decile）**；五等分称为**五分位组（Quintile）**。品种数量较少时，十分位组可能每组只有极少数品种，组合会过度集中。

### 14.2 等权与风险加权

**等权**：每个品种分配相同资金或名义敞口。

**波动率倒数加权**：波动越大的品种权重越小，使不同品种对组合风险的贡献更接近。

### 14.3 板块中性

对能源、金属、化工、农产品等板块分别控制多空敞口，避免策略无意中押注某个板块。

### 14.4 市场中性

市场中性通常指尽量降低对整体市场涨跌的暴露。多空名义金额相等不一定真正市场中性，因为不同品种的波动率和市场Beta不同。

### 14.5 名义敞口与净敞口

- **总敞口**：所有多头和空头名义价值绝对值之和；
- **净敞口**：多头名义价值减去空头名义价值。

一个多头50%、空头50%的组合净敞口约为0，总敞口约为100%，但仍然可能承担板块、波动率或Carry风险。

### 14.6 波动率目标

**波动率目标**是根据近期组合波动调整整体杠杆，使策略风险大致稳定。

市场剧烈波动时降低仓位，市场平稳时适度提高仓位。它不能消除突然跳空和涨跌停风险。

### 14.7 分散化

分散化是把风险分配给多个不完全同步的品种或因子。持有很多高度相关的黑色品种，并不等于真正分散。

### 14.8 完全抵押与超额收益

期货只占用保证金，但学术研究常假设组合是**完全抵押（Fully Collateralized）**的：为全部期货名义敞口预留等额资金，未占用部分投资于现金或低风险资产。

**超额收益（Excess Return）**是投资收益减去无风险利率或现金收益。夏普比率和因子Alpha通常关注超额收益。

完全抵押有助于让不同策略的杠杆口径可比较，也能避免把高杠杆误认为高投资能力。

### 14.9 仓位上限与风险预算

**仓位上限**限制单个品种或板块占组合的最大比例，避免一个判断错误摧毁整个组合。

**风险预算**则限制每个品种、板块或因子可以贡献多少组合风险。资金权重相同不代表风险贡献相同。

## 15. 回测表现指标

### 15.1 累计收益与年化收益

**累计收益**描述整个期间财富增长多少。

**年化收益**把不同长度样本转换为大致可比较的一年收益。不能把几天的高收益简单年化后当成长期可持续水平。

### 15.2 年化波动率

日频收益的年化波动率常近似为：

\[
\sigma_{annual}=\sigma_{daily}\sqrt{N}
\]

其中 \(N\) 为一年交易日数量。该公式依赖若干简化假设，遇到自相关和波动聚集时需谨慎。

### 15.3 夏普比率

**夏普比率（Sharpe Ratio）**衡量单位总波动对应的超额收益：

\[
Sharpe=\frac{E[R_p-R_f]}{\sigma(R_p-R_f)}
\]

夏普越高通常表示风险调整后表现越好，但它会受到样本长度、收益非正态、自相关和数据挖掘影响。

### 15.4 Sortino比率

**Sortino比率**只用下行波动作为风险分母，更关注不利波动。

### 15.5 最大回撤

**最大回撤（Maximum Drawdown）**是组合净值从某个历史高点到之后低点的最大跌幅。

它回答：“如果在最不幸的高点开始投资，之后最大可能经历多深的亏损？”

### 15.6 Calmar比率

\[
Calmar=\frac{\text{年化收益}}{|\text{最大回撤}|}
\]

它用最大回撤评价收益质量。

### 15.7 换手率

**换手率（Turnover）**描述组合持仓变化有多大。换手越高，通常交易成本越高。

每日重新排序、频繁切换合约的策略可能拥有漂亮的成本前收益，却无法在现实中实施。

### 15.8 手续费、买卖价差、滑点与价格冲击

- **手续费**：交易所和经纪机构收取的费用；
- **买卖价差**：立即买入需支付卖价，立即卖出只能得到买价；
- **滑点**：实际成交价比回测假设价格更差；
- **价格冲击**：订单较大时，交易本身推动市场价格。

这些共同构成交易摩擦。

### 15.9 容量

**策略容量（Capacity）**指在不显著降低收益的情况下，策略能够承载多大资金。

小资金在冷门品种上的高收益不一定能被大资金复制。

## 16. 样本内、样本外与可信研究

### 16.1 训练集、验证集和测试集

- **训练集**：估计模型参数；
- **验证集**：选择模型和超参数；
- **测试集**：最后一次评价从未参与选择的模型。

测试集一旦被反复查看并用于改模型，实际上就不再是真正测试集。

### 16.2 样本内与样本外

**样本内**是模型已经用来学习或选择的时期。

**样本外**是模型当时没有使用的未来时期。

金融市场会变化。历史拟合好并不意味着未来有效，所以样本外表现是本项目最核心的证据。

### 16.3 为什么不能随机打乱

普通机器学习经常随机划分数据，但金融数据有时间顺序。随机打乱后，模型可能用未来时期的信息帮助预测过去，形成不真实的优势。

### 16.4 滚动或扩展窗口

**扩展窗口**：训练数据随时间不断增加。

**滚动窗口**：只保留最近一段历史，让模型更适应市场变化。

示意：

```text
训练 2014-2018 → 验证 2019 → 测试 2020
训练 2014-2019 → 验证 2020 → 测试 2021
训练 2014-2020 → 验证 2021 → 测试 2022
```

### 16.5 重叠标签、Purging与Embargo

如果每一天都预测未来5日收益，相邻样本的标签会共享部分日期。例如周一和周二的“未来5日”高度重叠。

- **Purging**：删除训练和测试边界附近可能重叠的样本；
- **Embargo**：在训练和测试之间留出隔离期。

目的是降低标签重叠导致的信息渗透。

### 16.6 数据泄漏

**数据泄漏（Data Leakage）**是模型在预测时获得了现实中当时不可能知道的信息。

常见泄漏包括：

- 使用未来收益构造当前特征；
- 用全样本均值和标准差处理训练数据；
- 使用后来修订的库存或宏观数据；
- 当日收盘后才能计算的信号，却假设按同一收盘价成交；
- 用完整测试期选择因子方向；
- 先看全部回测，再只汇报最优参数。

### 16.7 前视偏差

**前视偏差（Look-ahead Bias）**是数据泄漏的一种，指使用未来信息做过去的决策。

代码中错误使用负向位移、居中窗口或未来最大最小值，都可能产生前视偏差。

### 16.8 生存者偏差

如果只选今天仍活跃的品种回到十年前做研究，就忽略了历史上退市、长期不活跃或失败的品种，这称为**生存者偏差**。

正确做法是让品种池随历史时间动态变化。

### 16.9 过拟合

**过拟合（Overfitting）**是模型记住了历史噪声，而不是学到可重复规律。

表现为：

- 训练期极好、测试期很差；
- 参数稍变，结果完全不同；
- 只在一个年份或一个板块有效；
- 模型复杂度远高于样本能够支持的程度。

### 16.10 数据挖掘与多重检验

如果尝试1,000个无效因子，也可能偶然找到几个历史表现很好的因子。只展示幸存者，会夸大真实能力。

常见控制方法包括：

- 预先登记候选因子和检验规则；
- 保留全部试验结果；
- 控制False Discovery Rate；
- 使用Deflated Sharpe Ratio；
- 设置完全未触碰的最终测试期。

### 16.11 Deflated Sharpe Ratio

**Deflated Sharpe Ratio（折减夏普比率）**尝试考虑：

- 研究者尝试了多少策略；
- 收益分布是否非正态；
- 样本长度是否足够。

它比直接报告最高夏普更能防范“从大量尝试中挑冠军”。

### 16.12 安慰剂检验

**安慰剂检验（Placebo Test）**人为破坏真实关系，例如：

- 随机打乱未来收益；
- 在品种之间随机置换因子；
- 使用经济上无关的随机特征。

如果模型在安慰剂数据上仍表现很好，通常意味着流程有泄漏、过拟合或评价错误。

## 17. 一次完整的概念性研究流程

以下只描述“研究在做什么”，不涉及如何编程。

### 第一步：确定每天可以投资的品种

排除刚上市、长期无成交、数据严重缺失或临近交割且不适合持有的合约。这个集合称为**投资池或品种池（Universe）**。

### 第二步：确定实际交易合约

为每个品种按照统一规则选择主力、近月或其他可交易合约，并规定何时换月。

### 第三步：计算因子

在每个日期、每个品种上计算动量、Carry、流动性、波动率、偏度、量价和产业链特征。

所有输入必须在决策时已经可得。

### 第四步：横截面处理

对因子做缩尾、排名、标准化和板块中性化，使不同品种和不同因子可比较。

### 第五步：定义预测目标

例如预测从下一交易日开始的未来5日收益，而不是从已经结束的当日收盘前开始计算。

### 第六步：检验单因子

计算Rank IC、ICIR、五分组收益和多空组合，筛掉没有稳定经济含义或样本外能力的因子。

### 第七步：比较模型

至少比较：

1. 单因子排序；
2. 等权多因子；
3. 线性或Elastic Net；
4. LightGBM或XGBoost。

只有机器学习稳定超过简单基准，复杂模型才有存在价值。

### 第八步：形成组合

根据预测分数买入排名靠前的品种、卖出排名靠后的品种，控制板块、单品种和整体风险。

### 第九步：按真实规则回测

使用实际合约，处理换月、涨跌停、不可成交、手续费和滑点，不能假设每次都以理想价格成交。

### 第十步：解释与压力测试

检查收益来自哪些因子、板块和年份；改变换月规则、预测期限和成本假设；设置安慰剂检验。

### 第十一步：得出有限结论

合理的结论不是“机器学习能够稳定赚钱”，而更可能是：

> 在指定样本、品种池、换月规则和成本假设下，某类机器学习模型是否相对简单基准提供了稳定的样本外增量；这种增量主要来自哪些因子，并在哪些市场状态下失效。

## 18. 一个极简的数字例子

假设某周一有五个商品品种。我们只使用动量和Carry两个因子：

| 品种 | 动量排名分 | Carry排名分 | 等权综合分 |
|---|---:|---:|---:|
| 铜 | 0.8 | 0.6 | 0.7 |
| 原油 | 1.0 | -0.2 | 0.4 |
| 豆粕 | -0.4 | 1.0 | 0.3 |
| 白糖 | 0.2 | -0.4 | -0.1 |
| 玻璃 | -0.8 | -1.0 | -0.9 |

最简单策略可能：

- 做多综合分最高的铜；
- 做空综合分最低的玻璃；
- 持有到下周调仓。

假设之后铜上涨2%，玻璃下跌1%，忽略权重细节时：

\[
R^{LS}\approx 2\%-(-1\%)=3\%
\]

若手续费、滑点和换月合计消耗0.4%，成本后约为2.6%。

但一次成功没有统计意义。真实研究需要多年、多个品种、多个滚动测试期，并检查亏损时期和极端风险。

机器学习的作用，就是不再机械地把动量和Carry等权相加，而是从历史训练期学习：

- 哪些因子更重要；
- 因子关系是否非线性；
- 动量与Carry组合时是否更有效；
- 不同市场状态下权重是否变化。

## 19. 如何阅读三篇参考资料

### 19.1 Hu等：Graph Portfolio / HCGNN

阅读时重点理解：

- 为什么把品种视为图节点；
- 产业链和替代关系如何形成边；
- 什么是时空图神经网络；
- 为什么同时设置移动平均、价格区间和变化点任务；
- 什么是持续学习和灾难性遗忘。

需要保留的怀疑：其数据约覆盖14天、测试期约4天。如此短的回测很难证明长期可投资性，极高累计收益不能直接外推。

### 19.2 Cheng、Liu、Zhou：LLM期货因子

阅读时重点理解：

- LLM如何根据字段生成候选公式；
- 单因子多空和多因子组合如何构造；
- 什么是IC、ICIR、Alpha和样本外；
- 为什么提示词、模型和温度会造成生成差异；
- 为什么必须检查未来信息和代码异常。

需要保留的怀疑：LLM训练截止日期之后的数据，不自动等于严格金融样本外。人类反复改提示词、选择表现好的因子，同样会产生数据挖掘偏差。

### 19.3 Bianchi等：商品期货曲线策略

阅读时重点理解：

- 同一商品为什么同时有多个到期合约；
- 什么是期货曲线；
- 水平、斜率和曲率分别表示什么；
- 跨期价差和蝶式价差怎样暴露于曲线变化；
- 为什么高换手策略必须做交易成本检验。

这篇文章提供的是国际商品市场证据，不能自动证明相同因子在中国市场也有效；本项目的价值正是进行本土化、样本外和成本后检验。

## 20. 初学者最容易产生的误解

### 误解1：预测准确率高就一定赚钱

错误。预测幅度、方向、交易成本、仓位和极端损失都会影响收益。

### 误解2：机器学习一定优于简单模型

错误。金融信号弱、噪声大、制度变化快。简单模型经常因为更稳定而在样本外胜出。

### 误解3：回测收益就是实际可获得收益

错误。实际交易还有价差、滑点、冲击、涨跌停、换月和容量限制。

### 误解4：连续合约就是可直接交易的永久合约

错误。连续合约是研究者拼接出的分析序列。

### 误解5：多空等金额就没有风险

错误。组合仍可能暴露于板块、波动率、流动性、Carry和相关性变化。

### 误解6：相关性就是产业链因果

错误。相关品种可能只是共同受到宏观或市场情绪影响。

### 误解7：LLM没有见过未来数据就没有数据泄漏

错误。公式、数据处理、提示词筛选和测试集使用都可能产生泄漏或数据挖掘。

### 误解8：只要夏普很高，策略就可靠

错误。短样本、反复试验、非正态收益和错误成本假设都能制造异常高夏普。

### 误解9：因子名字新颖就代表真正创新

错误。很多“新因子”只是动量、波动率和成交量的重新包装。

### 误解10：所有商品都可以使用完全相同的现货和库存数据

错误。不同品种的数据定义、公布频率、交割标准和产业结构差异很大。

| 期货 | 交易所标准化的未来月份合约 |
| 合约乘数 | 一手合约代表的商品数量 |
| 保证金 | 建立期货仓位时缴纳的风险担保资金 |
| 杠杆 | 用较少资金承担较大名义风险 |
| 做多 | 价格上涨时获利的方向 |
| 做空 | 价格下跌时获利的方向 |
| 主力合约 | 某品种中成交或持仓最活跃的合约 |
| 连续合约 | 把不同到期合约拼接出的研究序列 |
| 换月 | 从旧合约移仓到新合约 |
| 基差 | 现货和期货之间的价格差，符号需确认 |
| 期限结构 | 同一商品不同到期合约的价格曲线 |
| Contango | 远月通常高于近月的向上曲线 |
| Backwardation | 近月通常高于远月的向下曲线 |
| Carry | 期限结构包含的持有或展期收益线索 |
| 展期收益 | 换月及曲线变化带来的收益或损失 |
| 因子 | 用来解释或预测收益的量化特征 |
| 横截面 | 同一时点比较多个品种 |
| 时间序列 | 比较同一品种在不同时间的变化 |
| Rank IC | 因子排名与未来收益排名的相关性 |
| ICIR | IC平均水平相对于其波动的大小 |
| Alpha | 控制已知风险后仍无法解释的超额收益 |
| Beta | 对系统性风险的暴露 |
| 多空组合 | 买入高分组、卖出低分组的组合 |
| 中性化 | 去除板块或其他共同风险影响 |
| 换手率 | 组合持仓变化的频繁程度 |
| 滑点 | 实际成交价相对理想价格的损失 |
| 最大回撤 | 净值从历史高点到之后低点的最大跌幅 |
| 夏普比率 | 单位总波动对应的超额收益 |
| 样本内 | 用于训练或选择模型的数据时期 |
| 样本外 | 未参与模型训练和选择的未来时期 |
| 数据泄漏 | 使用现实中决策时不可得的信息 |
| 过拟合 | 学到历史噪声而非可重复规律 |
| GNN | 利用品种关系图进行学习的神经网络 |
| LLM因子 | 大语言模型提出的候选因子公式 |
| SHAP | 解释特征如何影响模型预测的方法 |
